# ggsegBrodmann: Brodmann Areas Atlas for the 'ggseg' Ecosystem

Brodmann areas atlas for the 'ggseg' ecosystem. Provides a unified
'ggseg_atlas' object with both 2D polygon geometry and 3D vertex
indices, for use with 'ggseg' and 'ggseg3d'.
